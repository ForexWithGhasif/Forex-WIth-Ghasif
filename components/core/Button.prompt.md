Premium action button — gold metallic is the hero CTA; emerald drives "join/convert" moments.

```jsx
<Button variant="primary" size="lg">Join VIP Signals</Button>
<Button variant="emerald" iconRight={<Icon name="arrow-up-right" />}>Start free</Button>
<Button variant="secondary">See the track record</Button>
<Button variant="ghost">Log in</Button>
<Button variant="outlineGold" as="a" href="#plans">View plans</Button>
```

Variants: `primary` (gold), `emerald` (success/convert), `secondary` (glass), `ghost`, `outlineGold`. Sizes `sm | md | lg`. Hover lifts + glow; press shrinks to 0.975. Supports `iconLeft`/`iconRight`, `fullWidth`, `disabled`, and `as="a"`.
